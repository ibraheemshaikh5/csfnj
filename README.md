# Care & Share Foundation NJ

[![Next.js](https://img.shields.io/badge/Next.js-16-black?logo=next.js)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.9-blue?logo=typescript)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.1-38bdf8?logo=tailwindcss)](https://tailwindcss.com/)
[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?logo=vercel)](https://vercel.com)

A modern, full-stack website with a visual CMS for **Care & Share Foundation**, a New Jersey-based charity dedicated to serving the community through food distribution, volunteer programs, and community outreach.

---

## Features

**Public Website**
- Responsive design optimized for mobile, tablet, and desktop
- Dynamic event pages with image galleries
- Volunteer registration and contact forms
- Donation integration with Zeffy, Venmo, and Zelle
- SEO optimized with proper metadata

**Admin CMS**
- Visual WYSIWYG editing — edit content exactly as it appears on the site
- Drag & drop reordering for impact items and events
- Real-time auto-save with instant feedback
- Image upload with drag & drop support
- Secure authentication with protected routes

## Tech Stack

| Category | Technology |
|----------|------------|
| **Framework** | Next.js 16 (App Router) |
| **Language** | TypeScript |
| **Database** | PostgreSQL via Supabase |
| **ORM** | Drizzle ORM |
| **Authentication** | Supabase Auth |
| **File Storage** | Supabase Storage |
| **Styling** | Tailwind CSS 4 |
| **Forms** | Formspree |
| **Donations** | Zeffy |
| **Analytics** | Vercel Analytics & Speed Insights |
| **Hosting** | Vercel |

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage with carousel, impact grid, events, and donation widget |
| `/about` | Organization story, mission, and video gallery |
| `/events` | Upcoming and past events listing |
| `/events/[slug]` | Individual event details with photo gallery |
| `/volunteer` | Volunteer registration form |
| `/contact` | Contact information |
| `/donate` | Donation portal with multiple payment methods |
| `/admin` | Visual CMS dashboard |
